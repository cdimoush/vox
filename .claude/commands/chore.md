# Chore Planning

Create a plan in specs/*.md to resolve the `Chore` using the `Plan Format` below.

## Instructions

- Research the codebase before planning. Start with `README.md`.
- Create the plan as `specs/<chore-name>.md`.
- Replace every `<placeholder>` with specific details.
- THINK HARD. Be thorough so nothing is missed.

## Relevant Files

- `README.md` - Project overview
- `scripts/` - Python scripts
- `.claude/commands/` - Slash commands
- `specs/` - Example specs

## Plan Format

```md
# Chore: <chore name>

## Chore Description
<describe the chore and why it's needed>

## Problem
<what issue does this chore address?>

## Solution
<how will this resolve the problem?>

## Relevant Files
### Files to Modify
<list files to change and why>

### New Files
<list any new files, if applicable>

## Step by Step Tasks
<list tasks as h3 headers with bullet points. Order matters. Last step: run Validation Commands.>

## Validation Commands
<commands to verify the chore is complete>

## Notes
<optional: additional context>
```
